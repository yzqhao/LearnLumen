# 项目概览
- 本项目是学习Unreal Engine的Lumen技术，全局光照技术
- Visual Studio项目的DirectX12工程，clang编译不过
- 关键路径：lumen/

# 代码结构
- math/ -> 数学库的封装
- common/ -> d3d12的一些封装
- lumen/ -> lumen实现，核心是LumenApp类
- lumen/shader/ -> 对应的shader目录
- res/ -> 资源文件，包括模型数据和SDF数据
- DX12.sln -> visual studio工程文件

# 关联仓库
- 无任何关联仓库，仅依赖d3d12渲染库
